1 # RMF #
2 
3 [![Build Status](https://travis-ci.org/salilab/rmf.png?branch=develop)](https://travis-ci.org/salilab/rmf)
4 
5 The library provides support for the RMF file format for
6 storing hierarchical molecular data (such as atomic or coarse grained
7 representations of proteins), along with markup, including geometry
8 and score data.
9 
10 If [HDF5](http://www.hdf5.org) is available, wrappers for it will be
11 built and support for older RMF-based formats will be compiled.
12 
13 See
14 - [RMF viewers](doc/Viewing.md) for more information about viewing RMF files,
15 - [RMF file format](\ref format) for more information about the files,
16 - [RMF Library](\ref library) for more information about using the library.
17 - [Mapping data into RMF files](\ref mapping) for more information about how to structure data in RMF files.
18 
19 Also see the [rmf examples](http://www.github.com/salilab/rmf_examples) repository
20 for examples of interesting or problematic RMF files.
