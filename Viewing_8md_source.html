9 # Chimera # {#chimera}
10 
11 The [Chimera](https://www.cgl.ucsf.edu/chimera/) research group has
12 provided support for loading and interacing with RMF files. Chimera
13 supports
14 - interactive display of restraints and restraint scores
15 - geometry
16 
17 
18 # VMD # {#vmd}
19 
20 We provide a plugin for
21 [VMD](http://www.ks.uiuc.edu/Research/vmd/). Instructions for how to
22 install it can be found on the [VMD plugin page](\ref vmdplugin). The
23 plugin supports
24 - multiresolution modules using the IMP::decorator::Alternative decorator
25 - multiple states stored using the IMP::decorator::State decorator
26 - fast loading
27 - displaying restraints as bonds
28 - static geometry
29 
30 # Pymol # {#pymol}
31 
32 We provide a plugin for Pymol. Instruction can be found at the
33 [Pymol plugin page](\ref pymolplugin). The plugin supports creating
34 multiple molecules from one RMF based on chains or RMF::decorator::Molecule
35 labels as well as dynamics geometry.
