Detailed Description

template<class Base>
class RMF::HDF5::ConstAttributes< Base >

Wrap an gettings the attributes from an HDF5 Object. Each data set or group can have associated data. These data should be relativel small (a few k at most). See MutableAttributes for the methods to change the attribute data.
