RMF Documentation

The library provides support for the RMF file format for storing hierarchical molecular data (such as atomic or coarse grained representations of proteins), along with markup, including geometry and score data.

If HDF5 is available, wrappers for it will be built and support for older RMF-based formats will be compiled.

See

  • RMF viewers for more information about viewing RMF files,
  • RMF file format for more information about the files,
  • RMF Library for more information about using the library.
  • Mapping data into RMF files for more information about how to structure data in RMF files.

Also see the rmf examples repository for examples of interesting or problematic RMF files.
